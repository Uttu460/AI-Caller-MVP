Quick-Start Guide

Follow these steps to fully configure your OutboundAI production platform:

  1. Make sure you run supabase_schema.sql inside your Supabase dashboard SQL Editor.
  2. Go to ⚙️ Settings, enter your LiveKit Cloud Credentials, then click save.
  3. Enter your Google Gemini API Key (Gemini Live preview models require v1alpha API key).
  4. Configure your Vobiz SIP Telephony details and click ⚡ Create SIP Trunk in LiveKit.
  5. Customize Olivia's flow in ✏️ AI Prompt or create customized agent profiles in 🤖 Agents.
  6. Launch test dials from 📞 Single Call.

Congratulations, your system is fully enterprise-ready and production-tested! 🚀
